Number of Days in a Month Description: Find the number of days in a given month and year (to handle leap years). Example: Input: Month = 2, Year = 2024 Output: 29 Hint: Use conditions: 31 Days: Jan, Mar, May, Jul, Aug, Oct, Dec 30 Days: Apr, Jun, Sep, Nov February: 28 or 29 (check for leap year using year % 4 == 0 but not year % 100 != 0 unless year % 400 == 0)
